Add shipping info to WooCommerce orders, change status to 'Completed' to send tracking details to Worten Marketplace (Mirakl), and mark as shipped.

== Description ==

Implement custom fields in WooCommerce orders to store the carrier name and tracking number, allowing for API shipment updates to accurately reflect shipping information on the Worten Marketplace (Mirakl) platform.
Only carriers that are registered in the Worten Marketplace will be available.

Note: Requires WooCommerce Installed on Your WordPress

== Instalation ==

1. Go to `Plugins` in Admin menu
2. Click `Add new`
3. Search for `Worten Conector - Tracking for orders` and click `Install now` or click in button `upload` to install `tracking-worten.zip`
4. Click in `Activate Plugin`
